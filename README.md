# Smart Energy Management System

## English | [中文](#中文)

A smart energy monitoring and management system built with Go, Gin, PostgreSQL, and Redis.

### Features
- Real-time energy consumption monitoring
- Smart meter data collection (IoT)
- Electricity/gas/water usage analytics
- Billing and tariff management
- Energy efficiency recommendations
- Peak shaving and load management
- Renewable energy tracking (solar, wind)
- Alerting and anomaly detection
- Carbon footprint calculation
- Building/zone-level monitoring

### Quick Start
```bash
docker-compose up -d
go run cmd/api/main.go
```
